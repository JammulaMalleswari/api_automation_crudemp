# api_automation_crudemp

## Project Description:
* API Testing Automation framework - Rest-assured, CRUD operations of an Employee api. 
* Written in Java with TestNG, RestAssured & Maven
* Can run test scripts in parallel

## Setup:
* Install Java 17
* Install Maven

## Run tests: 
* URL has to be updated in the config.properties file with valid key to get the expected response e.g `010d75835e9446f58f0aaf6379c88181`
* To execute the tests, run below command 
* `mvn clean verify`

## To Generate Report:
* To generate and open the report for the test run status, run below command
* `mvn allure:serve`
